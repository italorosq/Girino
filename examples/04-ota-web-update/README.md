# Exemplo 04 — Atualização OTA via Web

Demonstração de como usar OTA para atualizar o firmware sem cabo USB.

## Objetivos de Aprendizagem

- Entender o conceito de OTA (Over-The-Air)
- Configurar e usar ElegantOTA
- Atualizar firmware remotamente

## Conceitos

### OTA (Over-The-Air)

Permite atualizar o firmware do ESP8266 via Wi-Fi, sem conexão física. Útil para:
- Dispositivos instalados em locais de difícil acesso
- Atualização em lote de múltiplos dispositivos
- Manutenção remota em laboratório

### ElegantOTA

Biblioteca que fornece interface web para upload de firmware com:
- Barra de progresso em tempo real
- Suporte a firmware e filesystem
- Reinicialização automática após upload
